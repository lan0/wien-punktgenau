alle verfügbaren anzeigen
Diese Applikation soll vor allem Daten darstellen, die (noch) nicht im offiziellen Stadtplan aufscheinen, aber im Zuge der OGD Initiative freigegeben wurden.

Es ist außerdem möglich, durch einen Klick auf "alle verfügbaren anzeigen" alle (über den GeoServer) freigegebenen Daten anzuzeigen. Die Beschreibungen in den Popups sind dann allerdings nur mit den Rohdaten gefüllt und sehen eventuell nicht so toll aus (z.B. BAUMOGD statt Baumkataster oder BEZ statt Bezirk).

Hilfe

Daten

Auf der linken Seite können die anzuzeigenden Daten ausgewählt werden. Dabei können die Ebenen beliebig miteinander kombiniert werden, wobei nur ein Base-Layer (Karte, Satellit oder Flächenwidmung) gleichzeitig angezeigt werden kann. Je nach der Rechenleistung des Computers, was angezeigt wird und wie schnell der wien.gv.at Server antwortet, kann es einige Zeit dauern bis die Daten erscheinen.

Darstellung

Die Datensätze können auf zwei verschiedene Arten dargestellt werden:

Rastergrafiken

Bei der Rastergrafiken-Anzeige werden die Daten als gerasterte Bilder von wien.gv.at geladen. Da der Server die internen Vektordaten in Echtzeit umrechnet, kann es einige Zeit dauern bis der Server antwortet. Bei sehr vielen Daten (z.B. Haltestellen) ist diese Variante allerdings meistens schneller. Wenn diese Anzeigeart gewählt wurde, lassen sich keine Zusatzinformationen anzeigen.

Vektorbasiert

Die Daten werden als Vektordaten geladen und durch den Client auf die Karte gezeichnet. Sehr viele oder komplizierte Daten (z.B. stehende Gewässer) erfordern bei dieser Variante relativ viel Rechenleistung. Die maximale Anzahl der dargestellten Elemente kann angepasst werden, um Ladezeiten zu verkürzen. Sobald die Anzeige merklich ruckelt, sollten Sie entweder Rastergrafiken auswählen oder die Anzahl der angezeigten Elemente verkleinern. Durch einen Klick auf die dargestellten Daten können Zusatzinformationen angezeigt werden.

Permalink

Indem Sie die URL aus Ihrer Adresszeile kopieren, können Sie den derzeitigen Kartenausschnitt inklusive aktiver Ebenen speichern und verschicken.
